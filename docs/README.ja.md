## 説明

**Shadow Library** は、代替のゲーム認証方法を通じて Steam ユーザーの機能を拡張するソフトウェアソリューションです。このアプリケーションは、**[SteamTools](https://steamtools.net/)**および**GreenLuma**という人気のあるツールを通じて、ゲームのロック解除を管理するための便利な Web インターフェースを提供します。

---

## 機能

### 動作モード

| モード | 説明 |
|--------|------|
| **[SteamTools](https://steamtools.net/)** | ライセンスエミュレーションを通じてゲームをロック解除する人気のあるツール。[公式](https://steamtools.net/) サイトから [SteamTools](https://steamtools.net/) をダウンロードしてください |
| **GreenLuma** | Steam のチェックをバイパスする代替ソリューション |

### マニフェストソース

| ソース | 説明 |
|--------|------|
| **SteamAutoCrack / ManifestHub** | 自動更新付きの実際のマニフェスト |
| **ManifestAutoUpdate** | 代替マニフェストソース |

### 利点

- リポジトリからのマニフェストの自動ダウンロード
- 手動介入なしでロック解除ファイルの設定
- ダークテーマのモダンな Web インターフェース
- 多言語サポート（ロシア語、英語、中国語など）
- デポとツールの情報を含むロック解除されたゲームの履歴

## 要件

開始する前に、次のプログラムがインストールされていることを確認してください：

| 要件 | 説明 | リンク |
|------|------|--------|
| **SteamTools** | SteamTools モードに必要 | [steamtools.net](https://steamtools.net/) |
| **Steam** | インストールされた Steam クライアント | [store.steampowered.com](https://store.steampowered.com/) |
| **Python** | インストールされた現在のバージョンの Python | [python.org](https://www.python.org/downloads/) |

### SteamTools のインストール

1. 公式サイトにアクセス：**[SteamTools](https://steamtools.net/)**
2. 最新バージョンのプログラムをダウンロードしてインストール

## 使用手順

### 初期設定

**<u>[SteamTools](https://steamtools.net/) がインストールされていることを確認してください。インストールされていない場合、GreenLuma によるクラックのみが機能します</u>**

1. 初回起動時に、右上隅の**「Steam パス」**メニューを開きます
2. Steam ルートフォルダ（`steam.exe` がある場所）へのパスを指定
3. **「保存」**をクリック

### ゲームのロック解除

#### ステップ 1. ゲーム識別子を入力

入力フィールドに以下を指定：
- ゲームの **AppID**（数値、例：`2486820`）
- または Steam のゲームページへの**完全なリンク**

#### ステップ 2. ロック解除ツールを選択

| ツール | 説明 |
|--------|------|
| **SteamTools** | ほとんどのゲームに推奨。`config/stplug-in` フォルダに Lua スクリプトを作成 |
| **GreenLuma** | 代替方法。`config.vdf` を変更し、`AppList` にファイルを作成 |

#### ステップ 3. マニフェストソースを選択

| ソース | 説明 |
|--------|------|
| **SteamAutoCrack** | ManifestHub リポジトリからの実際のマニフェスト |
| **ManifestAutoUpdate** | 代替マニフェストソース |

#### ステップ 4. 実行

**「ゲームをロック解除」**ボタンをクリック

正常に処理されると、ゲームはロック解除されたゲームのリストに表示されます。

### ロック解除されたゲームの管理

| アクション | 説明 |
|------------|------|
| **リストを表示** | すべてのロック解除されたゲームはページの下部に表示されます |
| **削除** | 「削除」をクリックして変更を元に戻し、データベースからエントリを削除 |

## 制限と互換性

### サポートされているゲーム

- Steam DRM 保護付きのシングルプレイヤーゲーム
- パブリックアクセスのデポとマニフェストを持つゲーム

### 制限

| 制限 | 説明 |
|------|------|
| **Denuvo** | Shadow Library を通じてゲームをダウンロードできますが、Denuvo 保護により起動が防止されます — 追加のロック解除またはバイパスが必要 |
| **オンラインゲーム** | マルチプレイヤー機能は制限されています。ライセンスが Family Sharing を通じて利用可能な場合、オンラインは機能する可能性がありますが、常にではありません |
| **サードパーティランチャー** | 追加のランチャー（`EA App`、`Ubisoft Connect`、`Battle.net`）を必要とするゲームは、ほとんどの場合サポートされていません |

### Denuvo のバイパス

1. コンピュータでゲームの正規版を起動します — これにより Denuvo がデバイスを認証し、その後 Shadow Library を通じてクラック版を使用できます
2. ハイパーバイザーを介したバイパスの使用（追加ツールが必要）
