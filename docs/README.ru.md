## Описание

**Shadow Library** — это программное решение, расширяющее возможности пользователей Steam за счёт альтернативных методов активации игр. Приложение предоставляет удобный веб-интерфейс для управления разблокировкой игр через популярные инструменты: **[SteamTools](https://steamtools.net/)** и **GreenLuma**.

---

## Возможности

### Режимы работы

| Режим | Описание |
|-------|----------|
| **[SteamTools](https://steamtools.net/)** | Популярный инструмент для разблокировки игр через эмуляцию лицензии, для его работы скачайте [SteamTools](https://steamtools.net/) с [официального](https://steamtools.net/) сайта |
| **GreenLuma** | Альтернативное решение для обхода проверок Steam |

### Источники манифестов

| Источник | Описание |
|----------|----------|
| **SteamAutoCrack / ManifestHub** | Актуальные манифесты с автоматическим обновлением |
| **ManifestAutoUpdate** | Альтернативный источник манифестов |

### Преимущества

- Автоматическая загрузка манифестов из репозиториев
- Настройка файлов разблокировки без ручного вмешательства
- Современный веб-интерфейс с тёмной темой
- Поддержка нескольких языков (русский, английский, китайский и другие)
- История разблокированных игр с информацией о депотах и инструментах

## Инструкция по использованию

### Первоначальная настройка

**<u>Не забудьте проверить, установлен ли у вас</u>** **[<u>SteamTools</u>](https://steamtools.net/)**, <u>иначе будет работать только взлом через GreenLuma</u>

1. При первом запуске откройте меню **«Путь к Steam»** в правом верхнем углу
2. Укажите путь к корневой папке Steam (где расположен `steam.exe`)
3. Нажмите **«Сохранить»**

### Разблокировка игры

#### Шаг 1. Введите идентификатор игры

В поле ввода укажите:
- **AppID** игры (числовое значение, например `2486820`)
- Или **полную ссылку** на страницу игры в Steam

#### Шаг 2. Выберите инструмент разблокировки

| Инструмент | Описание |
|------------|----------|
| **SteamTools** | Рекомендуется для большинства игр. Создаёт Lua-скрипт в папке `config/stplug-in` |
| **GreenLuma** | Альтернативный метод. Модифицирует `config.vdf` и создаёт файлы в `AppList` |

#### Шаг 3. Выберите источник манифестов

| Источник | Описание |
|----------|----------|
| **SteamAutoCrack** | Актуальные манифесты из репозитория ManifestHub |
| **ManifestAutoUpdate** | Альтернативный источник манифестов |

#### Шаг 4. Запуск

Нажмите кнопку **«Разблокировать игру»**

После успешной обработки игра появится в списке разблокированных.

### Управление разблокированными играми

| Действие | Описание |
|----------|----------|
| **Просмотр списка** | Все разблокированные игры отображаются в нижней части страницы |
| **Удаление** | Нажмите кнопку «Удалить» для отката изменений и удаления записи из базы |

## Ограничения и совместимость

### Поддерживаемые игры

- Одиночные игры с защитой Steam DRM
- Игры с депотами и манифестами в открытом доступе

### Ограничения

| Ограничение | Описание |
|-------------|----------|
| **Denuvo** | Игры можно загрузить через Shadow Library, однако защита Denuvo предотвратит запуск — потребуется дополнительная разблокировка или обход защиты |
| **Онлайн игры** | Ограничена функциональность многопользовательских игр. Онлайн может работать, если лицензия доступна через Family Sharing, но не всегда |
| **Сторонние лаунчеры** | Игры, требующие дополнительные лаунчеры (`EA App`, `Ubisoft Connect`, `Battle.net`), в большинстве случаев не поддерживаются |

### Обходы для Denuvo

1. Запустите лицензионную версию игры на своём компьютере — это позволит Denuvo авторизовать устройство, после чего можно использовать взломанную версию через Shadow Library
2. Использование обходов через гипервизор (требует дополнительных инструментов)
