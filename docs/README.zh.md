## 描述

**Shadow Library** 是一个软件解决方案，通过替代游戏激活方法扩展 Steam 用户的功能。该应用程序通过流行工具 **[SteamTools](https://steamtools.net/)** 和 **GreenLuma** 提供方便的 Web 界面来管理游戏解锁。

---

## 功能

### 工作模式

| 模式 | 描述 |
|------|------|
| **[SteamTools](https://steamtools.net/)** | 通过许可证模拟解锁游戏的流行工具。从 [官方](https://steamtools.net/) 网站下载 [SteamTools](https://steamtools.net/) |
| **GreenLuma** | 绕过 Steam 检查的替代解决方案 |

### 清单来源

| 来源 | 描述 |
|------|------|
| **SteamAutoCrack / ManifestHub** | 带自动更新的实际清单 |
| **ManifestAutoUpdate** | 替代清单来源 |

### 优势

- 从仓库自动下载清单
- 无需手动干预即可配置解锁文件
- 带有深色主题的现代 Web 界面
- 多语言支持（俄语、英语、中文等）
- 已解锁游戏历史记录，包含仓库和工具信息

## 使用说明

### 初始设置

**<u>不要忘记检查是否安装了</u>** **[<u>SteamTools</u>](https://steamtools.net/)**，<u>否则只有 GreenLuma 破解能工作</u>

1. 首次启动时，打开右上角的 **「Steam 路径」**菜单
2. 指定 Steam 根文件夹的路径（`steam.exe` 所在位置）
3. 点击 **「保存」**

### 解锁游戏

#### 步骤 1. 输入游戏标识符

在输入框中指定：
- 游戏的 **AppID**（数值，例如 `2486820`）
- 或 Steam 游戏页面的**完整链接**

#### 步骤 2. 选择解锁工具

| 工具 | 描述 |
|------|------|
| **SteamTools** | 推荐用于大多数游戏。在 `config/stplug-in` 文件夹中创建 Lua 脚本 |
| **GreenLuma** | 替代方法。修改 `config.vdf` 并在 `AppList` 中创建文件 |

#### 步骤 3. 选择清单来源

| 来源 | 描述 |
|------|------|
| **SteamAutoCrack** | 来自 ManifestHub 仓库的实际清单 |
| **ManifestAutoUpdate** | 替代清单来源 |

#### 步骤 4. 运行

点击 **「解锁游戏」**按钮

成功处理后，游戏将出现在已解锁游戏列表中。

### 管理已解锁游戏

| 操作 | 描述 |
|------|------|
| **查看列表** | 所有已解锁游戏显示在页面底部 |
| **删除** | 点击「删除」以回滚更改并从数据库中删除条目 |

## 限制和兼容性

### 支持的游戏

- 带有 Steam DRM 保护的单人游戏
- 具有公共访问仓库和清单的游戏

### 限制

| 限制 | 描述 |
|------|------|
| **Denuvo** | 可以通过 Shadow Library 下载游戏，但 Denuvo 保护将阻止启动 — 需要额外解锁或绕过 |
| **在线游戏** | 多人游戏功能有限。如果通过家庭共享可用许可证，在线可能有效，但不总是 |
| **第三方启动器** | 需要额外启动器（`EA App`、`Ubisoft Connect`、`Battle.net`）的游戏在大多数情况下不受支持 |

### Denuvo 绕过方法

1. 在您的计算机上运行游戏的正版版本 — 这将允许 Denuvo 授权设备，之后您可以通过 Shadow Library 使用破解版本
2. 使用通过管理程序的绕过（需要额外工具）
