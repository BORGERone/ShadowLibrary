## 설명

**Shadow Library** 는 대체 게임 인증 방법을 통해 Steam 사용자의 기능을 확장하는 소프트웨어 솔루션입니다. 이 애플리케이션은 **SteamTools** 및 **GreenLuma** 라는 인기 있는 도구를 통해 게임 잠금 해제를 관리하기 위한 편리한 웹 인터페이스를 제공합니다.

## 기능

### 작동 모드

| 모드 | 설명 |
|------|------|
| **[SteamTools](https://steamtools.net/)** | 라이선스 에뮬레이션을 통해 게임을 잠금 해제하는 인기 있는 도구. [공식](https://steamtools.net/) 사이트에서 [SteamTools](https://steamtools.net/) 를 다운로드하세요 |
| **GreenLuma** | Steam 확인을 우회하는 대체 솔루션 |

### 매니페스트 소스

| 소스 | 설명 |
|------|------|
| **SteamAutoCrack / ManifestHub** | 자동 업데이트가 있는 실제 매니페스트 |
| **ManifestAutoUpdate** | 대체 매니페스트 소스 |

### 장점

- 리포지토리에서 매니페스트 자동 다운로드
- 수동 개입 없이 잠금 해제 파일 구성
- 다크 테마의 모던한 웹 인터페이스
- 다국어 지원 (러시아어, 영어, 중국어 등)
- 저장소 및 도구 정보가 포함된 잠금 해제된 게임 기록

## 사용 지침

### 초기 설정

**<u>반드시</u>** **[<u>SteamTools</u>](https://steamtools.net/)** <u>가 설치되어 있는지 확인하십시오. 설치되지 않은 경우 GreenLuma 크랙만 작동합니다</u>**

1. 첫 실행 시 오른쪽 상단의 **「Steam 경로」**메뉴를 엽니다
2. Steam 루트 폴더 (`steam.exe` 가 있는 곳) 경로를 지정합니다
3. **「저장」**클릭

### 게임 잠금 해제

#### 1 단계. 게임 식별자 입력

입력 필드에 다음을 지정:
- 게임의 **AppID** (숫자 값, 예: `2486820`)
- 또는 Steam 게임 페이지への**전체 링크**

#### 2 단계. 잠금 해제 도구 선택

| 도구 | 설명 |
|------|------|
| **SteamTools** | 대부분의 게임에 권장. `config/stplug-in` 폴더에 Lua 스크립트 생성 |
| **GreenLuma** | 대체 방법. `config.vdf` 수정 및 `AppList` 에 파일 생성 |

#### 3 단계. 매니페스트 소스 선택

| 소스 | 설명 |
|------|------|
| **SteamAutoCrack** | ManifestHub 리포지토리의 실제 매니페스트 |
| **ManifestAutoUpdate** | 대체 매니페스트 소스 |

#### 4 단계. 실행

**「게임 잠금 해제」**버튼 클릭

성공적으로 처리되면 게임은 잠금 해제된 게임 목록에 표시됩니다.

### 잠금 해제된 게임 관리

| 작업 | 설명 |
|------|------|
| **목록 보기** | 모든 잠금 해제된 게임은 페이지 하단에 표시됩니다 |
| **삭제** | 「삭제」를 클릭하여 변경 사항을 취소하고 데이터베이스에서 항목 제거 |

## 제한 사항 및 호환성

### 지원되는 게임

- Steam DRM 보호가 있는 싱글 플레이어 게임
- 공개 액세스 저장소 및 매니페스트가 있는 게임

### 제한 사항

| 제한 | 설명 |
|------|------|
| **Denuvo** | Shadow Library 를 통해 게임을 다운로드할 수 있지만 Denuvo 보호로 인해 시작이 방지됩니다 — 추가 잠금 해제 또는 우회 필요 |
| **온라인 게임** | 멀티플레이어 기능은 제한됩니다. 라이선스가 Family Sharing 을 통해 사용 가능한 경우 온라인이 작동할 수 있지만 항상 그런 것은 아닙니다 |
| **타사 런처** | 추가 런처 (`EA App`, `Ubisoft Connect`, `Battle.net`) 가 필요한 게임은 대부분 지원되지 않습니다 |

### Denuvo 우회

1. 컴퓨터에서 게임의 정품 버전을 실행합니다 — 이렇게 하면 Denuvo 가 장치를 인증한 후 Shadow Library 를 통해 크랙 버전을 사용할 수 있습니다
2. 하이퍼바이저를 통한 우회 사용 (추가 도구 필요)
