## Descrição

**Shadow Library** é uma solução de software que expande as capacidades dos usuários Steam através de métodos alternativos de ativação de jogos. O aplicativo fornece uma interface web conveniente para gerenciar desbloqueio de jogos através de ferramentas populares: **[SteamTools](https://steamtools.net/)** e **GreenLuma**.

## Recursos

### Modos de operação

| Modo | Descrição |
|------|-----------|
| **[SteamTools](https://steamtools.net/)** | Ferramenta popular para desbloquear jogos através de emulação de licença. Baixe [SteamTools](https://steamtools.net/) do site [oficial](https://steamtools.net/) |
| **GreenLuma** | Solução alternativa para contornar verificações Steam |

### Fontes de manifests

| Fonte | Descrição |
|-------|-----------|
| **SteamAutoCrack / ManifestHub** | Manifests atuais com atualizações automáticas |
| **ManifestAutoUpdate** | Fonte de manifests alternativa |

### Vantagens

- Download automático de manifests de repositórios
- Configuração de arquivos de desbloqueio sem intervenção manual
- Interface web moderna com tema escuro
- Suporte multi-idioma (russo, inglês, chinês e mais)
- Histórico de jogos desbloqueados com informações de depósitos e ferramentas

## Instruções de uso

### Configuração inicial

**<u>Não esqueça de verificar se você tem</u>** **[<u>SteamTools</u>](https://steamtools.net/)** <u>instalado, caso contrário apenas o crack GreenLuma funcionará</u>

1. Na primeira inicialização, abra o menu **«Caminho Steam»** no canto superior direito
2. Especifique o caminho para a pasta raiz do Steam (onde está `steam.exe`)
3. Clique em **«Salvar»**

### Desbloquear um jogo

#### Passo 1. Digite o identificador do jogo

No campo de entrada, especifique:
- **AppID** do jogo (valor numérico, por exemplo `2486820`)
- Ou **link completo** para a página do jogo no Steam

#### Passo 2. Selecione a ferramenta de desbloqueio

| Ferramenta | Descrição |
|------------|-----------|
| **SteamTools** | Recomendado para maioria dos jogos. Cria script Lua na pasta `config/stplug-in` |
| **GreenLuma** | Método alternativo. Modifica `config.vdf` e cria arquivos em `AppList` |

#### Passo 3. Selecione a fonte de manifests

| Fonte | Descrição |
|-------|-----------|
| **SteamAutoCrack** | Manifests atuais do repositório ManifestHub |
| **ManifestAutoUpdate** | Fonte de manifests alternativa |

#### Passo 4. Iniciar

Clique no botão **«Desbloquear jogo»**

Após processamento bem-sucedido, o jogo aparecerá na lista de jogos desbloqueados.

### Gerenciando jogos desbloqueados

| Ação | Descrição |
|------|-----------|
| **Ver lista** | Todos os jogos desbloqueados são exibidos na parte inferior da página |
| **Excluir** | Clique em «Excluir» para reverter alterações e remover entrada do banco de dados |

## Limitações e compatibilidade

### Jogos suportados

- Jogos single-player com proteção Steam DRM
- Jogos com depósitos e manifests em acesso público

### Limitações

| Limitação | Descrição |
|-----------|-----------|
| **Denuvo** | Jogos podem ser baixados via Shadow Library, mas proteção Denuvo impedirá inicialização — desbloqueio adicional ou bypass necessário |
| **Jogos online** | Funcionalidade multijogador é limitada. Online pode funcionar se licença estiver disponível via Family Sharing, mas nem sempre |
| **Launchers de terceiros** | Jogos que exigem launchers adicionais (`EA App`, `Ubisoft Connect`, `Battle.net`) não são suportados na maioria dos casos |

### Contornos Denuvo

1. Inicie a versão legítima do jogo no seu computador — isso permite ao Denuvo autorizar o dispositivo, após o que você pode usar a versão crackeada via Shadow Library
2. Usando bypasses via hipervisor (requer ferramentas adicionais)
