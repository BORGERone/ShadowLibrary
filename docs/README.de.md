## Beschreibung

**Shadow Library** ist eine Softwarelösung, die die Möglichkeiten von Steam-Benutzern durch alternative Spielaktivierungsmethoden erweitert. Die Anwendung bietet eine praktische Weboberfläche zur Verwaltung von Spiel-Freischaltungen über beliebte Tools: **[SteamTools](https://steamtools.net/)** und **GreenLuma**.

---

## Funktionen

### Betriebsmodi

| Modus | Beschreibung |
|-------|--------------|
| **[SteamTools](https://steamtools.net/)** | Beliebtes Tool zum Freischalten von Spielen durch Lizenzemulation. Laden Sie [SteamTools](https://steamtools.net/) von der [offiziellen](https://steamtools.net/) Website herunter |
| **GreenLuma** | Alternative Lösung zum Umgehen von Steam-Überprüfungen |

### Manifest-Quellen

| Quelle | Beschreibung |
|--------|--------------|
| **SteamAutoCrack / ManifestHub** | Aktuelle Manifests mit automatischen Updates |
| **ManifestAutoUpdate** | Alternative Manifest-Quelle |

### Vorteile

- Automatischer Manifest-Download aus Repositories
- Konfiguration von Freischaltdateien ohne manuelles Eingreifen
- Moderne Weboberfläche mit dunklem Theme
- Mehrsprachige Unterstützung (Russisch, Englisch, Chinesisch und mehr)
- Verlauf freigeschalteter Spiele mit Depot- und Tool-Informationen

Stellen Sie vor dem Start sicher, dass die folgenden Programme installiert sind:

| Anforderung | Beschreibung | Link |

## Gebrauchsanleitung

### Ersteinrichtung

**<u>Vergessen Sie nicht zu überprüfen, ob Sie</u>** **[<u>SteamTools</u>](https://steamtools.net/)** <u>installiert haben, sonst funktioniert nur der GreenLuma-Crack</u>

1. Öffnen Sie beim ersten Start das Menü **«Steam-Pfad»** in der oberen rechten Ecke
2. Geben Sie den Pfad zum Steam-Stammverzeichnis an (wo sich `steam.exe` befindet)
3. Klicken Sie auf **«Speichern»**

### Spiel freischalten

#### Schritt 1. Spiel-ID eingeben

Geben Sie im Eingabefeld an:
- **AppID** des Spiels (numerischer Wert, z.B. `2486820`)
- Oder **vollständigen Link** zur Spielseite in Steam

#### Schritt 2. Freischalt-Tool auswählen

| Tool | Beschreibung |
|------|--------------|
| **SteamTools** | Empfohlen für die meisten Spiele. Erstellt Lua-Skript im Ordner `config/stplug-in` |
| **GreenLuma** | Alternative Methode. Ändert `config.vdf` und erstellt Dateien in `AppList` |

#### Schritt 3. Manifest-Quelle auswählen

| Quelle | Beschreibung |
|--------|--------------|
| **SteamAutoCrack** | Aktuelle Manifests aus ManifestHub-Repository |
| **ManifestAutoUpdate** | Alternative Manifest-Quelle |

#### Schritt 4. Starten

Klicken Sie auf die Schaltfläche **«Spiel freischalten»**

Nach erfolgreicher Verarbeitung erscheint das Spiel in der Liste der freigeschalteten Spiele.

### Verwaltung freigeschalteter Spiele

| Aktion | Beschreibung |
|--------|--------------|
| **Liste anzeigen** | Alle freigeschalteten Spiele werden unten auf der Seite angezeigt |
| **Löschen** | Klicken Sie auf «Löschen», um Änderungen rückgängig zu machen und Eintrag aus Datenbank zu entfernen |

## Einschränkungen und Kompatibilität

### Unterstützte Spiele

- Einzelspieler-Spiele mit Steam DRM-Schutz
- Spiele mit Depots und Manifests im öffentlichen Zugriff

### Einschränkungen

| Einschränkung | Beschreibung |
|---------------|--------------|
| **Denuvo** | Spiele können über Shadow Library heruntergeladen werden, aber Denuvo-Schutz verhindert den Start — zusätzliche Freischaltung oder Umgehung erforderlich |
| **Online-Spiele** | Multiplayer-Funktionalität ist eingeschränkt. Online kann funktionieren, wenn Lizenz über Family Sharing verfügbar ist, aber nicht immer |
| **Dritt-Launcher** | Spiele, die zusätzliche Launcher erfordern (`EA App`, `Ubisoft Connect`, `Battle.net`), werden größtenteils nicht unterstützt |

### Denuvo-Umgehungen

1. Starten Sie die legitime Version des Spiels auf Ihrem Computer — dies ermöglicht Denuvo, das Gerät zu autorisieren, danach können Sie die gecrackte Version über Shadow Library verwenden
2. Verwendung von Hypervisor-Umgehungen (erfordert zusätzliche Tools)
