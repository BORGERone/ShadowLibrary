## Description

**Shadow Library** est une solution logicielle qui étend les capacités des utilisateurs de Steam grâce à des méthodes alternatives d'activation de jeux. L'application fournit une interface web pratique pour gérer le déblocage de jeux via des outils populaires : **[SteamTools](https://steamtools.net/)** et **GreenLuma**.

---

## Fonctionnalités

### Modes de fonctionnement

| Mode | Description |
|------|-------------|
| **[SteamTools](https://steamtools.net/)** | Outil populaire pour débloquer des jeux via l'émulation de licence. Téléchargez [SteamTools](https://steamtools.net/) depuis le site [officiel](https://steamtools.net/) |
| **GreenLuma** | Solution alternative pour contourner les vérifications Steam |

### Sources de manifests

| Source | Description |
|--------|-------------|
| **SteamAutoCrack / ManifestHub** | Manifests actuels avec mises à jour automatiques |
| **ManifestAutoUpdate** | Source de manifests alternative |

### Avantages

- Téléchargement automatique des manifests depuis les dépôts
- Configuration des fichiers de déblocage sans intervention manuelle
- Interface web moderne avec thème sombre
- Support multi-langues (russe, anglais, chinois et plus)
- Historique des jeux débloqués avec informations sur les dépôts et outils

## Instructions d'utilisation

### Configuration initiale

**<u>N'oubliez pas de vérifier si vous avez</u>** **[<u>SteamTools</u>](https://steamtools.net/)** <u>installé, sinon seul le crack GreenLuma fonctionnera</u>

1. Au premier lancement, ouvrez le menu **« Chemin Steam »** dans le coin supérieur droit
2. Spécifiez le chemin du dossier racine de Steam (où se trouve `steam.exe`)
3. Cliquez sur **« Enregistrer »**

### Débloquer un jeu

#### Étape 1. Entrez l'identifiant du jeu

Dans le champ de saisie, spécifiez :
- **AppID** du jeu (valeur numérique, par exemple `2486820`)
- Ou **lien complet** vers la page du jeu dans Steam

#### Étape 2. Sélectionnez l'outil de déblocage

| Outil | Description |
|-------|-------------|
| **SteamTools** | Recommandé pour la plupart des jeux. Crée un script Lua dans le dossier `config/stplug-in` |
| **GreenLuma** | Méthode alternative. Modifie `config.vdf` et crée des fichiers dans `AppList` |

#### Étape 3. Sélectionnez la source de manifests

| Source | Description |
|--------|-------------|
| **SteamAutoCrack** | Manifests actuels depuis le dépôt ManifestHub |
| **ManifestAutoUpdate** | Source de manifests alternative |

#### Étape 4. Lancer

Cliquez sur le bouton **« Débloquer le jeu »**

Après un traitement réussi, le jeu apparaîtra dans la liste des jeux débloqués.

### Gestion des jeux débloqués

| Action | Description |
|--------|-------------|
| **Voir la liste** | Tous les jeux débloqués sont affichés en bas de la page |
| **Supprimer** | Cliquez sur « Supprimer » pour annuler les changements et retirer l'entrée de la base |

## Limitations et compatibilité

### Jeux pris en charge

- Jeux solo avec protection Steam DRM
- Jeux avec dépôts et manifests en accès public

### Limitations

| Limitation | Description |
|------------|-------------|
| **Denuvo** | Les jeux peuvent être téléchargés via Shadow Library, mais la protection Denuvo empêchera le lancement — déblocage ou contournement supplémentaire requis |
| **Jeux en ligne** | La fonctionnalité multijoueur est limitée. En ligne peut fonctionner si la licence est disponible via Family Sharing, mais pas toujours |
| **Launchers tiers** | Les jeux nécessitant des launchers supplémentaires (`EA App`, `Ubisoft Connect`, `Battle.net`) ne sont généralement pas pris en charge |

### Contournements Denuvo

1. Lancez la version légitime du jeu sur votre ordinateur — cela permet à Denuvo d'autoriser l'appareil, après quoi vous pouvez utiliser la version crackée via Shadow Library
2. Utilisation de contournements par hyperviseur (nécessite des outils supplémentaires)
