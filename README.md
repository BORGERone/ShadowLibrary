<h1 align="center">Shadow Library</h1>

**Shadow Library** - это инструмент, предназначенный для расширения возможностей пользователей Steam, позволяющий получить доступ к играм, используя альтернативные методы активации.

## Возможности

*   **Поддержка нескольких режимов:**  GreenLuma и SteamTools.
*   **Интеграция с репозиториями:** SteamAutoCrack и ManifestAutoUpdate.
*   **Простой процесс активации:** Используйте AppID игры или ссылку на игру в Steam.

## Как использовать

1.  **Укажите путь к Steam:** При первом запуске необходимо указать путь к вашей установленной копии Steam.
2.  **Введите информацию об игре:** Введите AppID игры или ссылку на страницу игры в Steam.
3.  **Выберите режим активации:** Выберите GreenLuma или SteamTools. Если выбран SteamTools, укажите источник манифестов (SteamAutoCrack или ManifestAutoUpdate).
4.  **Разблокируйте игру:** Нажмите кнопку "Разблокировать игру".

## Ограничения

*   **Совместимость:**  Активация не гарантируется для всех игр.
*   **Онлайн-игры:**  Поддержка онлайн-игр ограничена.  Онлайн-функциональность может быть доступна, если лицензия игры доступна через семейную библиотеку.
*   **Denuvo:**  Denuvo в настоящее время не поддерживается.  Возможный обход: запуск игры на компьютере с аккаунта где лицензия приобретена, после чего использование взломанной версии игры, либо же использование обхода с помощью гипервизора.
